# Thiraphak Mascot Figma Assets

PNG assets separated from the approved mascot screenshot, prepared to replace the Pee Oi mascot set in Figma.

## Main Files

- `thiraphak-mascot-character-sheet.png` - overview sheet with contact sheet and hero character.
- `thiraphak-mascot-contact-sheet.png` - six-pose contact sheet.
- `thiraphak-mascot-hero.png` - single larger mascot pose.

## Individual Poses

- `thiraphak-mascot-welcome.png` - waving pose.
- `thiraphak-mascot-title-deed.png` - document/title deed pose.
- `thiraphak-mascot-map.png` - map/location pose.
- `thiraphak-mascot-mobile.png` - phone/photo evidence pose.
- `thiraphak-mascot-harvest.png` - sugarcane bundle pose.
- `thiraphak-mascot-approved.png` - approved document pose.

## Compatibility Aliases

These are duplicated from the closest available screenshot poses so the file set matches the old mascot naming shape:

- `thiraphak-mascot-planting.png` -> harvest pose.
- `thiraphak-mascot-residue.png` -> mobile/photo pose.
- `thiraphak-mascot-sell.png` -> approved document pose.

Raw source crops are kept in `raw-crops/`.
